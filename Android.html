Android Based Advanced Book Locator And Library Manager

The project entitled Android based advanced book locator and library manager is developed using Android using front end, Asp.net as a web technology and SQL server as a back end. Android platform has become more popular and it holds a maximum number of Users when compared to all other platform .Unlike our existing approaches, proposed Android Based library Booking App made up of both android and web platform. The aim of this project is to create an Android application for the college Library that has an enhanced feature like book searching, Book availability checking, location, book issue, return and request submission for stock alert. It is a one-stop solution for retrieving all the library book information like availability, book details, author details etc. With the help of web technology admin updates the book details in his/her college library system. Updating detail includes book name, author name, quantity etc. To use this system students have to register in this system .Student can access this system after registration. Student can search for books and all other information of a particular book. With the help of this project student can submit/subscribe a request for book availability alert which can be automatically expired within two days. Due to this process, this system automatically controls overwhelming of subscription alert. The subscription notifications of requested book are alerted via SMS to the student mobile number via web service module.
